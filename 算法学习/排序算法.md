# 简单的排序算法

## 冒泡排序
锟斤拷
```cpp
    for(int i = 0;i < n;i++){
        for(int j = 0;j < n-i-1;j++){
            temp = a[j];
            a[j] = a[j+1];
            a[j+1] = temp;
        }
    }
```

## 选择排序
锟斤拷
```cpp
    for(int i = 0;i < n;i++){
        for(int j = 0;j < n - i;j++){
            if(a[j] > max){
                max = a[j];
                index = j;
            }
        }
        a[index] = a[n-1-i];
        a[n-1-i] = max;
    }
```
