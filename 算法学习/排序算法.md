# 简单的排序算法

## 冒泡排序
锟斤拷
```cpp
    for(int i = 0;i < n;i++){
        for(int j = 0;j < n-i-1;j++){
            temp = a[j];
            a[j] = a[j+1];
            a[j+1] = temp;
        }
    }
```

## 选择排序
思路：每次遍历选择最大数或最小数，遍历后缩短下一次的遍历范围

```cpp
    for(int i = 0;i < n;i++){
        for(int j = 0;j < n - i;j++){
            if(a[j] > max){
                max = a[j];
                index = j;
            }
        }
        a[index] = a[n-1-i];
        a[n-1-i] = max;
    }
```


## 插入排序
思路：把待排序的数插入到已排好的序列中，类似于扑克牌的整理。  

```cpp
for(int i = 1;i < size(a);i++){
    for(int j = i;j > 0 ;j--){
        if(a[j] < a[j-1]){
            temp = a[j];
            a[j] = a[j-1];
            a[j-1] = temp;
        }
    }   
}
```


# 进阶排序算法（需要用到分治思想）

## 快速排序

思路：取一个基准元素，将数组划分成左边比基准元素小右边比基准元素大的部分，再将这两部分重复上述操作。   
   
*这里初学弄了好久，打完后找豆包要了注释*   

```cpp

// 快速排序函数：对数组a的[left, right]区间进行升序排序
void quick_sort(int *a, int left, int right) {
    // 递归终止条件：区间无元素或只有1个元素，无需排序
    if (left >= right) return; 
    
    // 定义临时数组存储左/右部分元素，大小为当前区间长度
    int pl[right+1-left];  // 存储小于等于基准的元素
    int pr[right+1-left];  // 存储大于基准的元素
    int base = a[left];    // 选取区间左边界为基准值
    int rn = 0;            // 右部分元素计数
    int ln = 0;            // 左部分元素计数
    
    // 遍历区间（跳过基准），按基准拆分元素到临时数组
    for (int i = left+1; i <= right; i++) {
        if (a[i] > base) {
            pr[rn] = a[i];  // 大于基准→右数组
            rn++;
        } else {
            pl[ln] = a[i];  // 小于等于基准→左数组
            ln++;
        }
    }
    
    int index;             // 记录基准值最终位置
    int idx = left;        // 回填数据的起始索引（当前区间左边界）
    
    // 第一步：回填左部分元素到原数组
    for (int i = 0; i < ln; i++) {
        a[idx++] = pl[i];
    }
    index = idx;           // 基准值的位置 = 左部分末尾+1
    a[index] = base;       // 第二步：回填基准值
    idx++;                 // 移动索引，准备回填右部分
    
    // 第三步：回填右部分元素到原数组
    for (int i = 0; i < rn; i++) {
        a[idx++] = pr[i];
    }
    
    // 递归排序左区间[left, 基准前]和右区间[基准后, right]
    quick_sort(a, left, index-1);
    quick_sort(a, index+1, right);
    
    return;
}

```



## 归并排序

思路：将数组从中间划分到最小，再将重中间划分的两数组按顺序放到信新数组，把新数组复制到旧数组上。   

```cpp
//治
void sort_func(int* a,int left,int right){
	int mid = (left + right)/2;
	int b[right+1];
	int i_l = left;
	int i_r = mid+1;
	int i = left;
	
	while(i_l <= mid && i_r <= right){
		if(a[i_l] < a[i_r]){
			b[i] = a[i_l];
			i++;
			i_l++;
			
		}else{
			b[i] = a[i_r];
			i++;
			i_r++;
		}
		
	}
	
	while(i_l <= mid){
		b[i] = a[i_l];
		i++;
		i_l++;
	}
	
	while(i_r <= right){
		b[i] = a[i_r];
		i++;
		i_r++;
	}
	
	for(int j = left;j <= right;j++){
		a[j] = b[j];
	}
}

//分
void func(int *a,int left,int right){
	
	int mid = (left + right)/2;
	if(left >= right)return ;
	
	func(a,left,mid);
	func(a,mid+1,right);
	
	sort_func(a,left,right);
}

```
