# STL之Set容器   

**set**集合每个元素确定，互异且无序


## 定义集合

`set<类型，比较器> s`

## 队列的相关函数

### 插入
`s.insert(元素)`

### 删除
`s.erase(元素)`

### 查找
`s.find(元素)`
返回迭代器

### 数

`s.count(元素)`
只会放回0或1（找到数了返回1，没有返回0）

## 大小
`s.size()`
返回元素个数

## 清除
`s.clear()`


## 判断是否为空
`s.empty()`


### 其他
`s.end()`

返回最后元素的迭代器
