

# memset 内存设置

`memset(指针，填充内容，字节数)`

指针指向数组的某一位，填充内容的类型要与数组元素一致，注意**字节数**，char占一个字节，int占4个字节。如果不清楚类型的字节数可以用`sizeof（类型）`。

*注意！：memset 是 C 语言的内存操作函数，只能安全用于POD 类型（如 int、char、数组等简单类型）*

# memcpy 内存拷贝

`void * memcpy(void * denstination,const void * Source,size_t num)`

第一个参数是将要被复制的数组，第二个是复制的数组，第三个是字节数（拷多少）

*注意！：同上，memcpy 也只能安全用于POD 类型（如 int、char、数组等简单类型）*
